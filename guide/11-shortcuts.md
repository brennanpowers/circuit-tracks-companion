# 11. Shortcuts — every combo in one place

The jam-night cheat sheet. Sections: views · editing · transport & performance ·
power-on gestures.

## Views & modes

| Combo | Does | Detail |
|---|---|---|
| `Shift + Note` | Expanded Note View (synths: 4-octave keyboard) / Expanded Drum View (pads 29–32 = live Drums 1–4) | [§3](03-synth-tracks.md) / [§4](04-drum-tracks.md) |
| `Shift + Gate` (or Gate ×2) | Micro Step View | [§6](06-sequencing.md) |
| `Shift + Pattern Settings` (or ×2) | Probability View | [§6](06-sequencing.md) |
| `Shift + FX` | Side Chain View | [§8](08-fx-mixer-sidechain.md) |
| `Shift + Patterns` | View Lock on/off | [§7](07-patterns-scenes-arrangement.md) |
| `Shift + Save` | Setup View (exit with `Play`) | [§5](05-midi-tracks.md) |
| `Shift + Projects` | Packs View (needs microSD) | [§2](02-data-model.md) |
| Long-press any view button | Peek at that view while held | [§1](01-orientation.md) |
| Press `Shift` in Setup View | Sticky Shift on/off | [§1](01-orientation.md) |

## Editing

| Combo | Does | Detail |
|---|---|---|
| Hold step pad + keyboard pad | Add / remove that note on the step | [§6](06-sequencing.md) |
| `Clear` + step pad | Delete all notes on the step | [§6](06-sequencing.md) |
| `Duplicate` + source step + dest step(s) | Copy step with all attributes | [§6](06-sequencing.md) |
| `Shift + Velocity` | Fixed Velocity (96) on/off | [§6](06-sequencing.md) |
| `Shift + Record` | Rec Quantise on/off (hold `Shift` to read state) | [§6](06-sequencing.md) |
| `Shift + Duplicate` | **Mutate** the playing pattern (no undo!) | [§6](06-sequencing.md) |
| Hold sample pad + step pads | Sample flip (drum steps turn pink) | [§4](04-drum-tracks.md) |
| `Clear` + turn a macro ≥20% | Delete that macro's automation (whole pattern) | [§8](08-fx-mixer-sidechain.md) |
| `Clear + Step Page` | Pattern back to 16 steps (17–32 data kept) | [§7](07-patterns-scenes-arrangement.md) |
| `Duplicate + Step Page` | Extend to 32 steps, copying 1–16 → 17–32 | [§7](07-patterns-scenes-arrangement.md) |
| `Shift` + step pad (Pattern Settings View) | Move pattern start point | [§7](07-patterns-scenes-arrangement.md) |
| `Shift + ▼/▲` | Transpose whole pattern by octave | [§3](03-synth-tracks.md) |
| `Shift` + patch pad (Patch View) | Select patch silently (no preview) | [§3](03-synth-tracks.md) |
| Pad 24 in Micro Step View | Tie-forward (drone) on/off for the step | [§6](06-sequencing.md) |

## Transport & performance

| Combo | Does | Detail |
|---|---|---|
| `Shift + Play` | Continue from where you stopped (plain `Play` restarts at step 1) | [§6](06-sequencing.md) |
| `Shift` + pattern pad | Switch pattern instantly (plain tap = queued) | [§7](07-patterns-scenes-arrangement.md) |
| `Shift` + scene pad | Switch scene instantly (plain tap = queued at Drum 1 pattern end) | [§7](07-patterns-scenes-arrangement.md) |
| `Shift` + project pad | Switch project instantly (plain tap = queued) | [§2](02-data-model.md) |
| Hold first pattern/scene pad + press last | Chain patterns (contiguous) / scenes | [§7](07-patterns-scenes-arrangement.md) |
| Hold `Shift` + tap `Tempo/Swing` ×3+ | Tap tempo (averages last 5) | [§9](09-live-performance.md) |
| `Shift + Clear` | Click on/off (Clear green = on) | [§6](06-sequencing.md) |
| Hold `Record` > ~½ s | Momentary record: punch out on release | [§6](06-sequencing.md) |
| Mute a track, then its step pads | Play that track live over the mix | [§9](09-live-performance.md) |
| Macro 1 after first `Save` press | Choose project colour (14 options) | [§2](02-data-model.md) |

## Power-on gestures (hold while switching on)

| Hold | Does | Detail |
|---|---|---|
| `Shift` | Advanced Setup View: `Note` = Easy Start/mass storage, `Duplicate` = MIDI Thru→2nd Out, `FX` = master compressor. Exit with `Play` | [§8](08-fx-mixer-sidechain.md), [§10](10-components.md) |
| `Shift + Save` | Save Lock on/off (persists across power cycles) | [§2](02-data-model.md) |
| `Shift + Clear` | Force-load a blank project (recovery) | [§2](02-data-model.md) |
| `Scales + Preset + Note` | Bootloader Mode (firmware version/recovery; exit with `Play`) | [§10](10-components.md) |
